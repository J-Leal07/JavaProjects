# Exercícios de Lógica em Java

Repositório com **exercícios de lógica básica em Java**.

Os exercícios estão na pasta **exercicios** e as resoluções estão em arquivos `.java`.

---

## Como executar

Para rodar os códigos, é necessário:
- Ter uma **JVM (JDK)** instalada
- Usar uma **IDE que suporte Java**

Abra o projeto na IDE e execute o arquivo desejado.
